# Gestão de Sistemas e Redes



Este projeto consiste na configuração e gestão de uma infraestrutura de rede com equipamentos Cisco, aplicando na prática os principais conceitos de redes de computadores. Inclui a interligação de redes através de rotas estáticas, segmentação da rede com VLANs, encaminhamento entre VLANs, funcionamento de switches, implementação de redundância com HSRP, agregação de ligações com EtherChannel, utilização do CDP e uso de TFTP para copiar e repor as configurações. Este trabalho demonstra competências práticas em planeamento, configuração e administração de redes.
